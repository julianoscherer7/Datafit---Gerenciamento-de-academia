# Datafit---Gerenciamento-de-academia

DATAFIT – Escopo Resumido do Projeto
Descrição Geral

O DATAFIT é um sistema integrado de treinos, saúde e gestão de alunos. O projeto combina funcionalidades voltadas para usuários comuns (hipertrofia e bem-estar), praticantes de atividade física e também professores/academias que desejam organizar treinos e acompanhar alunos.

O sistema terá versões web e mobile, permitindo registro diário de treinos, controle de evolução, acompanhamento de medidas corporais e administração de treinos por profissionais.

Objetivos do Projeto

Registrar treinos, exercícios, cargas e séries executadas.

Exibir evolução do usuário através de históricos e gráficos.

Acompanhar saúde física por meio de medidas corporais e metas.

Permitir que professores cadastrem alunos e montem treinos personalizados.

Oferecer acesso rápido via web e mobile.

Tecnologias Utilizadas

Backend: Python (Flask ou FastAPI)

Banco de Dados: MySQL

Frontend Web: HTML, CSS, JavaScript

Aplicativo Mobile: Flutter

Ferramentas Extras:

SQL avançado (views, procedures e triggers)

Autenticação por tokens

Chart.js para gráficos

Principais Funcionalidades
1. Módulo Usuário

Cadastro e login

Registro de treinos, séries, cargas e repetições

Histórico diário, semanal e mensal

Gráficos de progresso

2. Módulo Saúde

Registro de medidas corporais (peso, peito, braço, cintura etc.)

Relatórios simples de evolução física

3. Módulo Academia

Cadastro de professores e alunos

Criação e edição de treinos personalizados

Acompanhamento de desempenho dos alunos

Pontos de Aprendizado Alinhados ao Curso de Banco de Dados

Modelagem relacional completa

Criação de views, stored procedures, triggers e consultas analíticas

Integração eficiente entre Python e MySQL

Construção de API REST estruturada

Desenvolvimento multiplataforma (web + mobile)
